Browser based Etch-a-Sketch

Key features and contributions:

  • Used modular functions with single responsibilities.
  • Implemented an interactive drawing grid using pure JavaScript DOM manipulation.
  • Implemented a flexible grid sizing (4x4 to 100x100) with automatic cell size calculation.
  • Utilised event listeners for hover effects (mouseenter).
  • Implemented a "Reset Sketch Pad" feature with an interactive reset button and user input handling.
  • Implemented number parsing and validation (4-100 range) and providing user feedback for invalid inputs.

This project demonstrates my understanding of fundamental JavaScript concepts with functions, loops, and event listeners, while showing understanding of input validation and error handling. It also shows my skills in DOM manipulation and event handling through dynamic grid generation and interactive hover effects.
